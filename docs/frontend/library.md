# ライブラリパネル (`library.js`)

`frontend/js/library.js` — ダウンロード済みトラックの検索・表示・操作を担当します。

## 主な機能

### トラック検索

テキスト入力によるリアルタイム検索を提供します。

**動作**

- 入力のたびに `api.listTracks({ search: query })` を呼び出す
- タイトルおよびアーティスト名で部分一致検索（バックエンド側で処理）

### トラックグリッド

検索結果をグリッド形式で表示します。

**表示情報**

| 項目 | 説明 |
|------|------|
| サムネイル | `thumbnail_url` から `<img>` 表示（なければプレースホルダー） |
| タイトル | `track.title` |
| アーティスト | `track.artist` |
| 再生時間 | `duration_secs` を `MM:SS` 形式に変換 |
| ダウンロードボタン | ファイルをブラウザにダウンロード（`download_url` を使用） |
| 削除ボタン | トラックを DB・ファイルから削除 |

**クリック動作**

トラックをクリックすると `player.js` にトラックを渡して再生開始します。

### トラック削除

`api.deleteTrack(id, true)` を呼び出し、DB レコードと音声ファイル・サムネイルをディスクから削除します。  
削除後にトラックリストを更新します。

### トラックファイルダウンロード

`download_url`（`/api/v1/files/{id}/download`）を `<a>` タグの `href` に設定してクリックイベントを発火します。  
`Content-Disposition: attachment` により `{title}.{format}` ファイル名でダウンロードされます。

## ページロード・ページネーション

1. 初回読み込みで `api.listTracks({ limit: 50, offset: 0 })` を実行
2. 取得結果をグリッドにレンダリング
3. 取得件数が `PAGE_SIZE`（50）件の場合、「もっと読み込む」ボタンを表示
4. ボタンクリックで `offset` をずらして追加取得・グリッドに追記

検索時は `offset` を 0 にリセットしてページネーションをやり直します。
