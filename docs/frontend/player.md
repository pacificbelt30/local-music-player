# プレイヤー (`player.js`)

`frontend/js/player.js` — 画面下部に固定表示されるオーディオプレイヤーを担当します。

## UI 構成

```
┌──────────────────────────────────────────────────────┐
│ [サムネイル]  タイトル            ◁ prev  ▶ play  ▷ next │
│              アーティスト                              │
│         ────────────●──────────────  02:15 / 03:33   │
└──────────────────────────────────────────────────────┘
```

## 主な機能

### トラック再生

外部モジュール（`library.js`・`queue.js`）からトラックオブジェクトを受け取って再生を開始します。

**再生時の動作**

1. `track.stream_url`（`/api/v1/stream/{id}`）を `<audio>` 要素の `src` に設定
2. `audio.play()` を呼び出す
3. サムネイル・タイトル・アーティストを表示更新
4. バックエンドが `play_count` と `last_played_at` を自動更新

### コントロール

| ボタン | 動作 |
|--------|------|
| 再生/一時停止 | `audio.play()` / `audio.pause()` のトグル |
| 前の曲 | プレイキュー内の前のトラックを再生 |
| 次の曲 | プレイキュー内の次のトラックを再生 |

### シークバー

`<input type="range">` で現在位置を操作します。

**動作**

- `audio.timeupdate` イベントで現在位置を更新
- ドラッグ操作で `audio.currentTime` を変更
- バックエンドが HTTP Range リクエストに対応しているためシーク可能

### 時間表示

```
MM:SS / MM:SS  （現在位置 / 総再生時間）
```

### プレイキュー

ライブラリパネルから複数トラックを選択すると、プレイヤーがキューとして管理します。曲終了時に自動的に次のトラックに進みます。

## イベント

| イベント | 動作 |
|---------|------|
| `audio.ended` | 次のトラックを自動再生 |
| `audio.error` | エラー表示 |
| `audio.loadedmetadata` | 総再生時間を表示 |

## 外部インターフェース

他のモジュールから以下の関数を呼び出します:

```javascript
// library.js / queue.js から
player.play(track);           // トラックを再生
player.queue(tracks);         // プレイキューをセット
player.addToQueue(track);     // キューに追加
```
